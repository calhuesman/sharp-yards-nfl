Sharp Yards is actively maintained and improved based on player feedback. If you've found incorrect player data, run into a bug, have an idea for a new feature, or just want to say hello, reach out by email:

What's helpful to include

If you're reporting a bug, it helps to include what you were doing when it happened, what device or browser you were using, and a screenshot if you have one. If you spotted an incorrect stat for a specific player, let us know the player's name and season — we double-check every correction against the original source data before making a fix.

Sharp Yards is a fan-made stat guessing game. Not affiliated with the NFL or any NFL team.
